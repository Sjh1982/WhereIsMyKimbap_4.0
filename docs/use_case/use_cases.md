# Use Case 목록

| ID | Use Case | 주 액터 | 주요 흐름 |
|---|---|---|---|
| UC-01 | 로그인 | 사용자 | 아이디·비밀번호 입력 → 해시 비교 → 권한별 메뉴 표시 |
| UC-02 | 주변 편의점 조회 | 사용자 | 위도·경도 입력 → 거리 계산 → 반경 2km 매장 정렬 |
| UC-03 | 매장 상품·재고 조회 | 사용자 | 매장·카테고리·상품명 선택 → 4개 테이블 JOIN → 가격·재고 표시 |
| UC-04 | 상품별 매장 비교 | 사용자 | 상품명 입력 → products 기준 LEFT JOIN → 미취급 상품 포함 표시 |
| UC-05 | 장바구니 관리 | 사용자 | 한 매장의 상품 추가 → 수량 변경/삭제 → 재고 검증 |
| UC-06 | 주문·가상 결제 | 사용자 | 장바구니 검증 → 주문/상세/결제 저장 → 재고 차감 → 바코드 발급 |
| UC-07 | 픽업 완료 | 사용자 | 대기 주문 선택 → 상태와 수령시각 갱신 |
| UC-08 | 상품·재고 CRUD | 관리자 | 상품 생성/삭제, 매장-상품 재고 관계 생성/수정/삭제 |

결제와 픽업 바코드는 과제용 시뮬레이션이며 외부 결제망 또는 POS와 연결되지 않는다.

# 구현 전 UI 와이어프레임

와이어프레임은 실제 실행 화면 캡처가 아니라 구현 전에 화면 요소와 사용자 흐름을 정의한 저충실도 설계다.

1. 주변 매장: 좌표 입력, 거리순 조회, 매장 카드.
2. 매장 상품: 편의점/카테고리 Dropdown, 상품 검색, 가격·재고·담기.
3. 상품 비교: 상품명 검색, 매장별 가격·재고, 미등록 행.
4. 장바구니: 수량 증감·삭제, 합계, 결제 방법.
5. 주문 내역: 주문 상태, 픽업 바코드, 수령 완료.
6. 데이터 관리: 상품과 재고 관계 CRUD 입력 영역.

PNG 파일은 이 폴더의 `wireframe_*.png`이며 구현 장의 실제 화면과 구분해 사용한다.
